# Sorting Tutorials
Simple Sorting Tutorials

## Prerequisite
Make sure you have installed:
1. [node js](https://nodejs.org/en/)
2. [npm](https://www.npmjs.com/get-npm)

## Dependency
Client dependency in `package.json`

## How To Use
In this directory (root), run:
```
$ npm i
$ npm run serve
```
